# Demo Samples

Bu klasore sunum icin en az 3 adet karsi-duygu ornegi koyun.

Onerilen dosya isimleri:

- calm_words_high_stress.wav
- positive_words_angry_tone.wav
- neutral_words_controlled_tone.wav

Notlar:

- RAVDESS veya EMO-DB uzerinden alinmis ornekler iyi calisir.
- Streamlit arayuzu bu klasorde bulunan wav/mp3/flac/ogg dosyalarini otomatik listeler.
- Kontrollu demo icin ayni klasore `demo_cases.json` koyabilirsiniz; script bu dosyadaki semantik prompt ve stres ipucunu kullanarak AcoSemantic kararini daha net gosterebilir.
